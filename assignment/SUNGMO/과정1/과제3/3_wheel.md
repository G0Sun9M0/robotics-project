# 바퀴(Wheel) 조사 보고서

> 로봇 운반차를 움직이는 기구 부품인 **바퀴**의 구조·재질, 작용하는 힘(마찰력·관성력), 구름 저항을 조사한 문서.
> ※ 작성 전 각 출처의 신뢰성·정확성을 교차 검증하였다. (검증 근거는 맨 아래 "출처 및 신뢰성" 참조)

## 1. 바퀴의 구조와 재질

### 1-1. 구조
바퀴는 크게 **휠(차륜 본체)** 과 **타이어(접지부)** 로 나뉜다.

| 부분 | 구성 요소 | 역할 |
|---|---|---|
| 휠(Wheel) | 허브(Hub) | 축(axle)과 바퀴를 연결, 회전 중심 |
| | 림(Rim) | 타이어를 끼워 지지하는 테두리 |
| | 스포크(Spoke) | 허브와 림을 연결, 하중 분산 |
| 타이어(Tire) | 트레드(Tread) | 노면과 닿는 바깥면 — **접지·견인(traction)** 담당, 패턴으로 배수·그립 조절 |
| | 사이드월(Sidewall) | 옆면 — 구조 지지·완충·규격 표시 |
| | 벨트(Belt) | 트레드 아래 보강층 — 형상 유지·내구 |
| | 비드(Bead) | 림에 밀착 고정 |
| | 이너라이너(Inner liner) | 튜브리스 타이어의 공기 밀폐층 |

### 1-2. 재질
- **휠(림)** : 강철, 알루미늄 합금, 카본 등 — 강성·경량 요구에 따라 선택
- **타이어** : 천연/합성 고무 + 섬유·강선·카본블랙 (공압 타이어 기준)
- **로봇 소형 바퀴** : 솔리드 고무, TPU/플라스틱, 폼(foam), 옴니휠/메카넘휠용 소형 롤러 등 — 무게·접지·내구를 고려

## 2. 바퀴에 작용하는 힘 — 마찰력과 관성력

### 2-1. 마찰력 (정지 마찰력 · 운동 마찰력)
- **정지 마찰력(Static friction)** : 두 표면이 서로 **미끄러지지 않을 때** 작용. 보통 운동 마찰보다 크다.
- **운동 마찰력(Kinetic friction)** : 두 표면이 **상대적으로 미끄러질 때** 작용.

**바퀴에 작용하는 방법 (핵심)**
- 정상적으로 **구르는(미끄러지지 않는) 바퀴**는 접지점이 노면에 대해 **순간적으로 정지** 상태다 → 바퀴를 앞으로 밀어 주는 **견인력(traction)은 "정지 마찰력"** 이다.
- 가속·제동 토크가 정지 마찰의 한계를 넘으면 바퀴가 **헛돌거나 미끄러짐(slip)** → 이때부터 **운동 마찰력**이 작용하고 그립을 잃는다. (빙판 가속 시 바퀴 헛도는 현상)
- 즉, 견인을 유지하려면 구동력이 **정지 마찰의 최대치 이내**여야 한다.

### 2-2. 관성력 (Inertia)
- **관성** : 물체가 현재의 운동 상태를 유지하려는 성질 (뉴턴 제1법칙). 질량이 클수록 크다.
- **바퀴에 작용하는 방법**
  - 직선 운동 관성 : 정지한 운반차를 출발시키거나(가속), 달리는 차를 멈추려(제동) 할 때 관성이 저항으로 작용 → 더 큰 견인(정지 마찰)·제동력이 필요.
  - 회전 관성(관성 모멘트) : 바퀴 자체를 돌리고 멈추는 데도 저항 → 무겁거나 큰 바퀴일수록 가감속이 둔해진다.
- 설계 시사 : 바퀴·차체가 무거우면 관성↑ → 출발·정지에 더 큰 토크와 마찰(그립)이 필요.

## 3. 구름 저항 (Rolling Resistance)

### 3-1. 개념
**구르는 물체(바퀴·타이어)가 노면 위를 굴러갈 때 진행을 방해하는 저항력.** (rolling friction/drag)
미끄럼 마찰과는 **다른 개념** — 미끄러지지 않아도 발생한다.

### 3-2. 발생하는 이유 — 변형 히스테리시스
- 바퀴가 하중을 받아 구르면, 접지부가 **눌렸다(압축) → 떨어지며 복원**되는 변형을 반복한다.
- 이때 **변형에 들어간 에너지보다 복원되는 에너지가 작다**(비탄성). 이 차이가 **열로 소실**되며, 이 손실이 곧 구름 저항이다. → 이를 **히스테리시스(hysteresis)** 라 한다. (공압 타이어 구름 저항의 약 **90%**)
- 부수 요인 : 노면의 영구 변형(무른 바닥의 바퀴 자국), 접지부의 미세 슬립 등.

### 3-3. 영향 인자 (요약)
- 타이어/노면 변형이 클수록 ↑ (공기압 낮으면 ↑), 단단한 노면일수록 ↓
- 하중이 클수록 ↑

> ⚠️ 보완(검증 중 확인) : "약 90%가 히스테리시스"는 **공압(空壓) 타이어** 기준 수치다. 로봇 소형 솔리드 바퀴는 변형 양상이 다르지만, "**변형-복원 손실(히스테리시스)이 주원인**"이라는 원리는 동일하게 적용된다.
