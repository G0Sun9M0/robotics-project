# 운반 로봇 제어 실습 (계산) 보고서

> 주어진 로봇 스펙으로 ① 센서 기반 경로 유지 제어 절차와 ② 엔코더 측정값 해석(이동·회전 계산)을 푼 문서.

## 0. 주어진 스펙과 기본 계산값

| 항목 | 값 |
|---|---|
| 바퀴 지름 D | 10 cm |
| 바퀴 둘레 C = πD | π × 10 = **31.416 cm** |
| 엔코더 해상도 | 1회전당 30 펄스 |
| **펄스당 이동거리** = C / 30 | 31.416 / 30 = **1.0472 cm/펄스** |
| 두 바퀴 간 거리(휠베이스) L | 20 cm |
| IR 센서 | 5 cm 앞 경로 감지, 3개 어레이 |
| 센서 값 규약 | 흰색 = 1, 검은색(경로) = 0 |

**가정**
- 바퀴 미끄러짐(slip) 없음, 좌우 바퀴 지름 동일, 노면 평탄 (엔코더 펄스 = 실제 이동에 비례).
- 1 펄스 = 1 카운트(별도 4체배 언급 없으므로 해상도 30을 그대로 1회전당 카운트로 사용).
- IR 센서 어레이 순서 = **[왼쪽, 중앙, 오른쪽]**.

## 문제 ①: 센서값 000 → 100, 경로 유지 제어 절차

### 상황 해석
- **000** : 세 센서 모두 검은색(0) = 모두 경로 위 → **정상(경로 중앙)** 주행 중.
- **100** : 왼쪽=1(흰색, 경로 이탈), 중앙=0(검정), 오른쪽=0(검정).

```
   [왼][중][우]
000  ■  ■  ■    ← 세 센서 모두 경로(검정) 위 = 정상
100  □  ■  ■    ← 왼쪽만 흰색으로 벗어남
        ↑ 경로(검정)는 중앙·오른쪽 아래에 있음 = 경로가 로봇 기준 "오른쪽"에 치우침
```

왼쪽 센서만 흰색이 되었다 = 로봇이 경로에 대해 **왼쪽으로 치우쳤다**(= 경로가 로봇의 오른쪽에 있다). → 다시 중앙(000)으로 오려면 **오른쪽으로 조향**해야 한다.

### 제어 절차 (단계별)
1. **센서 읽기**: IR 3개 어레이에서 `[1, 0, 0]` 측정.
2. **상태 판정**: 왼쪽 센서 경로 이탈, 중앙·오른쪽은 경로 위 → 로봇이 왼쪽으로 치우침(경로가 오른쪽).
3. **오차 계산**: 목표는 "경로가 중앙(000)". 현재 오차 = 경로가 오른쪽으로 +만큼 벗어남 → **오른쪽 보정 필요**.
4. **제어량 결정 (차동 구동)**: 오른쪽으로 회전 → **왼쪽 바퀴 속도 ↑, 오른쪽 바퀴 속도 ↓**.
   - 단순(on-off) 제어: 고정량만큼 우회전. PID 제어: 오차 크기에 비례해 조향량 결정(부드러움).
5. **모터 명령 출력**: 모터 드라이버로 좌·우 바퀴 속도 차 명령 → 살짝 우회전.
6. **재측정·반복(피드백 루프)**: 다시 센서를 읽어 `000`(중앙 복귀)이 될 때까지 2~5 반복.
7. **복귀 처리**: `000` 복귀 시 좌·우 속도를 같게 하여 **직진** 재개.

> 핵심: "이탈한 센서의 반대쪽 = 경로가 있는 쪽으로 조향"한다. 100(왼쪽 이탈) → 경로는 오른쪽 → **우회전**.

## 문제 ②: 엔코더 측정값으로 이동 설명 + 계산

### 측정 구간
| 구간 | 왼쪽 펄스 | 오른쪽 펄스 | 의미 |
|---|---|---|---|
| 1 | 100 | 100 | 좌우 동일 → 직진 |
| 2 | 20 | 10 | 왼쪽이 더 많이 회전 → 회전(우회전) |
| 3 | 50 | 50 | 좌우 동일 → 직진 |

### 구간 1 — 직진
- 각 바퀴 이동거리 = 100 × 1.0472 = **104.72 cm**
- 좌우 동일하므로 회전 없이 **직진 104.72 cm**.

### 구간 2 — 회전 (핵심 계산)
- 왼쪽 바퀴 거리 d_L = 20 × 1.0472 = **20.944 cm**
- 오른쪽 바퀴 거리 d_R = 10 × 1.0472 = **10.472 cm**
- 로봇 중심 이동거리 d_c = (d_L + d_R) / 2 = (20.944 + 10.472)/2 = **15.708 cm**

**회전 각도 θ** = (d_L − d_R) / L = (20.944 − 10.472) / 20 = 10.472 / 20 = **0.5236 rad**
$$θ = 0.5236 \text{ rad} × \frac{180}{\pi} = 30.0°$$

**회전 반경 R** = (L/2) × (d_L + d_R) / (d_L − d_R) = (20/2) × 31.416 / 10.472 = 10 × 3.0 = **30 cm**
- 검산: R = d_c / θ = 15.708 / 0.5236 = 30 cm ✓

**회전 방향**: 왼쪽 바퀴가 더 빠름(많이 회전) → **오른쪽으로 회전(시계방향)**.
- 회전 중심(ICC)은 로봇 오른쪽 30 cm 지점. 오른쪽 바퀴가 안쪽(반경 R−L/2 = 20 cm), 왼쪽 바퀴가 바깥쪽(반경 R+L/2 = 40 cm).
  - 검산: 안쪽 20cm × 0.5236 = 10.47cm = d_R ✓ / 바깥쪽 40cm × 0.5236 = 20.94cm = d_L ✓

### 구간 3 — 직진
- 각 바퀴 이동거리 = 50 × 1.0472 = **52.36 cm**
- 좌우 동일 → 회전한 방향(초기 대비 오른쪽 30° 틀어진 방향)으로 **직진 52.36 cm**.

### 종합 설명
로봇은 **① 직진 104.72 cm → ② 반경 30 cm로 오른쪽 30° 회전(중심 15.71 cm 이동) → ③ 새 방향으로 직진 52.36 cm** 의 경로를 그렸다.

| 항목 | 값 |
|---|---|
| 구간1 직진 | 104.72 cm |
| 구간2 회전(중심 이동) | 15.71 cm (반경 30 cm, **오른쪽 30°**) |
| 구간3 직진 | 52.36 cm |
| **로봇 중심이 이동한 총 경로 길이** | 104.72 + 15.71 + 52.36 = **약 172.79 cm** |
| **최종 방향 변화** | 초기 대비 **오른쪽으로 30° 회전** |

> 결론: 로봇은 총 약 **172.8 cm**를 이동했고, 중간에 **반경 30 cm의 호를 따라 오른쪽으로 30°** 회전했다. (회전은 구간 2에서만 발생, 구간 1·3은 직진)
