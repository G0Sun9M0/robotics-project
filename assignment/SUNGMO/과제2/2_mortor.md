# 모터(Motor) 조사 보고서

> 핵심 구동 부품인 모터의 유형·사양·제어 원리·드라이버를 조사한 문서.
> ※ 본 문서의 내용은 작성 전 각 출처의 신뢰성·정확성을 교차 검증하여 정리하였다. (검증 근거는 맨 아래 "출처 및 신뢰성" 참조)

## 1. 모터의 기본 유형

### 1-1. DC 모터
- **원리** : 자기장 속 도선에 전류가 흐르면 힘(로런츠 힘)이 발생하여 회전력(토크)이 생긴다.
- **배선** : 2선 (전원 +, 접지 −) — 전압을 걸면 **연속으로 회전**한다.
- **특징** : 구조가 단순하고 저렴, 빠른 연속 회전에 적합. 다만 **위치·정밀 제어에는 약함**.
- **용도** : 바퀴 구동, 팬, 펌프 등.

### 1-2. 서보 모터 (취미용/RC 기준)
- **구성** : 〔DC 모터 + 감속 기어 + 위치 센서(가변저항) + 제어 회로〕의 **조립체**.
- **배선** : 3선 (전원, 접지, **제어 신호**).
- **원리** : 목표 각도 신호를 주면, 내부 센서가 현재 위치를 되먹임(**피드백**)하여 오차를 줄이는 **폐루프 제어**로 정확한 각도에 멈춘다.
- **특징** : **각도(위치) 정밀 제어**에 강함. 취미용 서보는 보통 **0~180° 범위**로 회전이 제한된다.
- **용도** : 로봇 팔/관절, 조향, 카메라 짐벌 등.

> ⚠️ 보완(검증 중 확인) : 위 설명은 **취미용(RC) 서보** 기준이다. **산업용 AC 서보 모터**는 엔코더 피드백을 쓰며 **연속 회전**이 가능한 별개 범주이므로, "서보 = 180° 제한"은 취미용에 한정된 설명이다.

### (참고) 스텝 모터
- 펄스 한 번에 일정 각도씩 회전 → 개루프로도 위치 제어 가능. DC·서보와 함께 자주 비교되는 유형.

## 2. 모터의 사양을 구성하는 항목

| 사양 | 의미 |
|---|---|
| 정격 전압 / 전압 범위 | 정상 동작 전압과 허용 범위 (예: 6V 모터, 동작 3~9V) |
| 회전 속도 (RPM) | 분당 회전수. **반드시 전압과 함께 표기** (예: 2400 RPM @ 3V) |
| 무부하 속도 (No-load Speed) | 축에 아무것도 안 달았을 때의 최고 속도 |
| 정격 토크 (Rated Torque) | 정상 사용 시 축에 내는 회전력 |
| 스톨 토크 (Stall Torque) | **축이 멈춘 순간 낼 수 있는 최대 토크** (이 상태 지속 시 모터 손상 주의) |
| 정격 전류 / 스톨 전류 | 동작 전류와 정지 시 최대 전류 |
| 정격 출력(W) | 일률 |

**사양 간 관계**
- 속도는 **전압에 비례** (부하 고정 시): 전압↑ → 속도↑
- 토크는 **전류에 비례** : 전류↑ → 토크↑  (T = Kt · I)
- 전력 관계 : **P = V × I**

## 3. 모터의 제어 원리 — 속도와 방향

### 3-1. 회전 속도 제어 — PWM
- **PWM(펄스 폭 변조)** : 전압을 매우 빠르게 ON/OFF 반복하며, ON 시간의 비율(**듀티 사이클**)로 **평균 전압**을 조절한다.
- ON이 길수록(듀티↑) 평균 전압↑ → **빠르게**, ON이 짧을수록 평균 전압↓ → **느리게**. (0~100%)

### 3-2. 회전 방향 제어 — H-브리지
- **H-브리지** : 4개의 스위치를 "H"자로 배치하고 가운데 모터를 둔 회로. 스위치 조합으로 모터에 흐르는 **전류 방향을 반전**시켜 정/역회전을 만든다.
- 두 입력이 서로 반대 → 정회전 / 역회전, **두 입력이 같으면 정지**.

> PWM(속도) + H-브리지(방향)을 **함께** 쓰면 모터의 속도와 방향을 독립적으로 완전히 제어할 수 있다.

## 4. 모터 드라이버

- **필요 이유** : MCU(Arduino 등)의 출력 신호는 전류가 약해 모터를 **직접 구동할 수 없다**. 모터 드라이버가 약한 제어 신호를 받아 모터에 필요한 **큰 전류·전압으로 증폭**한다.
- **대표 예 : L298N** (듀얼 H-브리지 — DC 모터 2개 동시 제어)

| 핀 | 역할 |
|---|---|
| IN1, IN2 | 모터 A **방향** 제어 |
| IN3, IN4 | 모터 B **방향** 제어 |
| ENA, ENB | 각 모터의 **속도(PWM)** 제어 |

**L298N 방향 진리표 (모터 A 예)**

| IN1 | IN2 | 동작 |
|---|---|---|
| LOW | LOW | 정지 |
| HIGH | LOW | 정회전 |
| LOW | HIGH | 역회전 |
| HIGH | HIGH | 정지 |

→ ENA에 PWM을 넣어 속도를, IN1·IN2 조합으로 방향을 정한다. (방향=H브리지, 속도=PWM 원리의 실제 구현)
