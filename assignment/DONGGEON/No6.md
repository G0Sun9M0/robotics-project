# 차동 구동 방식 및 로봇 제어 기초 이해

## 1. 차동 구동 방식 (Differential Drive)
차동 구동 방식은 두 개의 독립적인 구동 바퀴를 사용하여 로봇의 이동을 제어하는 방식입니다. 좌우 바퀴의 회전 속도 차이를 이용해 방향을 전환하며, 구조가 단순하고 제어가 용이하여 모바일 로봇(AGV, AMR)에 널리 사용됩니다.

### 기본 원리
* **직진:** 좌우 바퀴의 회전 속도가 동일할 때 ($v_L = v_R$).
* **회전:** 좌우 바퀴의 회전 속도가 다를 때 ($v_L 
eq v_R$).
* **제자리 회전:** 양쪽 바퀴를 반대 방향으로 같은 속도로 회전시킬 때 ($v_L = -v_R$).

## 2. 각속도와 회전 반경
이동 로봇의 주행 경로를 결정하는 핵심 물리량입니다.

* **각속도 ($\omega$):** 단위 시간 동안 로봇이 회전하는 정도. 
  $$\omega = rac{v_R - v_L}{L}$$ (L은 휠 베이스)
* **회전 반경 ($R$):** 로봇 중심에서 회전 중심(ICR)까지의 거리.
  $$R = rac{v}{\omega} = rac{L}{2} \cdot rac{v_R + v_L}{|v_R - v_L|}$$

## 3. 휠 베이스($L$)의 영향
* **민감도:** 휠 베이스($L$)가 작을수록 적은 속도 차이로도 급격한 회전이 가능합니다.
* **정밀도:** 실제 제작 시 휠 베이스 값은 오도메트리(Odometry) 계산의 기준이 되므로 정확한 실측이 중요합니다.
